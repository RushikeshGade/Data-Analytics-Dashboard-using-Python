# 📖 Project Overview

The **Student Data Analytics Dashboard** is a Python-based data analysis project that performs SQL-like analytical operations on student records using **Pandas** and visualizes insights using **Matplotlib**.

The application loads student data from a CSV file, analyzes academic performance, attendance, fees, and demographic information, then generates informative reports and charts to support data-driven decision-making.

---

# 🎯 Objectives

* Analyze student academic performance.
* Generate course-wise and city-wise reports.
* Visualize student statistics.
* Perform SQL-like operations using Pandas.
* Build an interactive analytics workflow with Python.
